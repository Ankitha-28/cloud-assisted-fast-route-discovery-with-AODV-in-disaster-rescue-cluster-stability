# Cloud Assisted Fast Route Discovery in AODV for Disaster Rescue Cluster Stability

## Overview

This project proposes a Cloud Assisted Fast Route Discovery mechanism integrated with the Ad hoc On-Demand Distance Vector (AODV) routing protocol to improve communication in disaster rescue environments. The system uses Mobile Ad Hoc Networks (MANETs) to enable communication when traditional network infrastructure is unavailable or damaged.

## Problem Statement

During disasters such as earthquakes, floods, and cyclones, communication infrastructure often fails. Traditional AODV routing suffers from slow route discovery, frequent link failures, and unstable communication due to dynamic network conditions. These limitations can delay rescue operations and reduce coordination among rescue teams.

## Objectives

* Improve route discovery speed in MANET environments.
* Enhance communication reliability during disaster situations.
* Reduce routing overhead and communication delay.
* Improve cluster stability by selecting reliable nodes.
* Compare the performance of AODV and DSDV routing protocols.
